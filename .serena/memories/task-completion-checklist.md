# タスク完了時のチェックリスト

## コード品質
- [ ] **Lint チェック**: `npm run lint` でエラーなし
- [ ] **フォーマット**: `npm run format` で全ファイルをフォーマット済み
- [ ] **型チェック**: TypeScript の型エラーなし
- [ ] **コメント**: 複雑なロジックに日本語コメントを追加

## テスト
- [ ] **ユニットテスト**: 新機能に対応するテストを追加
- [ ] **テスト実行**: `npm test` で全テスト成功
- [ ] **カバレッジ**: 重要な機能のカバレッジを確認（`npm run test:coverage`）

## ビルド
- [ ] **ビルド成功**: `npm run build` でエラーなし
- [ ] **dist/確認**: 生成されたファイルに問題なし

## ドキュメント
- [ ] **README更新**: 新機能を追加した場合、README.mdを更新
- [ ] **JSDoc**: 公開APIにはJSDocコメントを追加
- [ ] **サンプル**: 必要に応じてexamples/にサンプルを追加

## コミット前
- [ ] **変更内容確認**: `git status` と `git diff` で変更を確認
- [ ] **コミットメッセージ**: 日本語で明確に記述
- [ ] **不要なファイル**: デバッグ用ファイルなどをコミットしない

## リリース前（該当する場合）
- [ ] **prepublishOnly**: `npm run prepublishOnly` でエラーなし
- [ ] **バージョン更新**: package.json のバージョンを更新
- [ ] **CHANGELOG**: 変更内容を記録
